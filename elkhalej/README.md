# Elkhalej Car Rental | شركة الخليج لتأجير السيارات

Welcome to the official repository of **Elkhalej Car Rental**, a premium car rental application built with a modern, dark-themed interface and a seamless user experience.

أهلاً بكم في المستودع الرسمي لشركة **الخليج لتأجير السيارات**، تطبيق تأجير سيارات متميز بواجهة عصرية وتجربة مستخدم فريدة.

## 🚀 Key Features | المميزات الرئيسية

-   **Multi-Service Fleet**: Dedicated sections for Airport Transfers, Trips, Weddings, and Business travel.
-   **Premium UI**: Dark mode with glassmorphism effects, smooth animations, and a responsive layout.
-   **Localized Experience**: Full support for Arabic and English branding.
-   **Fast Booking**: One-click booking via WhatsApp for immediate confirmation.
-   **Dynamic Catalog**: Real-time car data sync from local mocks (ready for Google Sheets integration).

## 📂 Project Structure | هيكل المشروع

-   `index.html`: The main landing page with a premium splash screen.
-   `airport.html`, `trips.html`, `wedding.html`, `business.html`: Specialized service pages.
-   `style.css`: Modern styling with CSS variables and responsive breakpoints.
-   `data.js`: Core data management logic.
-   `script.js`: Frontend interactivity and booking management.
-   `competitor_data.js`: Local car fleet and pricing data synchronized from top competitors.

## 🛠️ How to Run | طريقة التشغيل

1.  Clone the repository.
2.  Open `index.html` in any modern web browser.
3.  (Optional) Run a local HTTP server for the best experience:
    ```bash
    npx http-server .
    ```
