# منصة حساب البصمة الكربونية 🌍
**مهندسون من أجل مصر المستدامة (ESE-2030)**

تم تصميم وتطوير هذه المنصة الذكية لمساعدة المؤسسات بمختلف قطاعاتها (صناعي، تجاري، طبي وغيرها) على حساب البصمة الكربونية السنوية الخاصة بها، وتقييم انبعاثات الغازات الدفيئة الناتجة عن استهلاكاتهم المختلفة للكهرباء، والوقود، والمياه، بالإضافة إلى طرق إدارتهم للمخلفات.

المنصة مصممة لتوفير تقارير دقيقة وموثوقة مدعومة برسوم بيانية تفصيلية، مع توجيه النصائح والتوصيات البيئية لدعم استراتيجيات الاستدامة المؤسسية (ESG).

## 🌟 المميزات الأساسية للبرنامج (Features)
1. **واجهة مستخدم حديثة ومتجاوبة:** تدعم اللغة العربية (RTL) بشكل أصيل، ومصممة لتعمل بسلاسة على أجهزة الكمبيوتر والهواتف الذكية.
2. **آلة حاسبة ذكية متعددة الخطوات (Multi-step Calculator):** تتيح إدخال البيانات بطريقة منظمة ومقسمة إلى قطاعات:
   - البيانات الأساسية.
   - استهلاك الكهرباء (بحسب فواتير الكيلووات).
   - استهلاك الوقود والانتقالات (سولار، بنزين، غاز طبيعي).
   - إدارة المخلفات (المخلفات العامة، الطبية/الخطرة، والمُعاد تدويرها).
   - استهلاك المياه.
3. **تحليل فوري للانبعاثات:** استخدام معاملات التحويل البيئية القياسية (Emission Factors) لتحويل الاستهلاكات إلى ما يعادلها من كيلوجرامات من ثاني أكسيد الكربون (kg CO₂e).
4. **لوحة تحكم ديناميكية (Dashboard):** 
   - تلخيص كلي لإجمالي الانبعاثات التقديرية (بالكيلوجرام وبالطن المتري).
   - تحديد "أعلى مصدر للانبعاثات" بشكل آلي.
   - عرض رسم بياني تفاعلي (Bar Chart) لتوضيح نسبة الانبعاثات في كل قطاع.
5. **توصيات ذكية مخصصة:** يقدم النظام توصيات بيئية وتخفيفية بناءً على أعلى مصدر للانبعاثات (مثال: ترشيد الطاقة إن كانت الكهرباء هي الأعلى، أو إدارة أسطول النقل إن كان الوقود هو الأعلى).
6. **تصدير وطباعة التقارير (PDF Export):** إمكانية طباعة تقرير البصمة الكربونية في صفحة واحدة منسقة ورسمية، تحتوي على شعار المنظمة، والتحليلات، والتوصيات لتقديمها للجهات المعنية.
7. **قاعدة بيانات افتراضية (Local Storage MVP):** لحفظ التقارير واسترجاعها للمستخدم.

## 🛠️ التقنيات المستخدمة (Technology Stack)
* **إطار العمل الأساسي (Framework):** React.js (Vite)
* **توجيه الصفحات (Routing):** React Router DOM (HashRouter)
* **مكتبة الرسوم البيانية (Charts):** Recharts
* **الأيقونات (Icons):** Lucide-React
* **إدارة الحزم:** npm
* **الاستضافة (Deployment):** متوافق مع GitHub Pages / Vercel باستخدام تقنية Single Page Application (SPA).

## 🚀 كيفية تشغيل المشروع محلياً (Local Development)

### المتطلبات المسبقة:
يجب أن يكون لديك `Node.js` مثبتاً على جهازك.

### الخطوات:
1. قم بتحميل كود المشروع أو عمل `Clone` للمستودع.
2. افتح موجه الأوامر (Terminal) داخل مجلد المشروع.
3. قم بتثبيت الحزم البرمجية بكتابة الأمر التالي:
```bash
npm install
```
4. لتشغيل خادم التطوير ورؤية المنصة، اكتب:
```bash
npm run dev
```
5. افتح الرابط الذي سيظهر في الشاشة (عادةً `http://localhost:5173`) لرؤية الموقع.

## 📦 كيفية رفع المشروع أونلاين (Deployment)
المشروع معد ليتم رفعه مباشرة باستخدام `GitHub Actions` على `GitHub Pages`.
يتم الأمر عبر دفع الكود (Push) لفرع `main` حيث سيقوم ملف `.github/workflows/deploy.yml` ببناء مجلد `dist` وتشغيله أوتوماتيكياً.
